Sample models

Sample models are distributed under the same licence as gEcon package. Licence terms can be found here.

In order to run those models save them (.gcn files) on disk, load gEcon package under R, and run the accompanying R scripts (in the same directory, after setting R working directory there).

For information about the language and the R interface please refer to the Users' guide, which can be found under the documentation tab.
