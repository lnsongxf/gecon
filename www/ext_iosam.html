gEcon.iosam package

Calibration of large-scale CGE and DSGE models using Input-Output Tables and  Social Accounting Matrices can become a difficult and tedious task. gEcon.iosam package is designed to assist gEcon users in this task by providing iosam class for representing Input-Output Tables and Social Accounting Matrices and a set of functions for importing and manipulating them. To streamline the process of calibration of CGE (and multisector DSGE) models written using gEcon template mechanism, the function get_flow_values is provided.

Installing gEcon.iosam package

Please note that gEcon.iosam is NOT DISTRIBUTED THROUGH CRAN. You have to download the .zip (Windows), .tgz (Mac), or .tag.gz (source, Linux / Un*x) file, save it on your disk, and install gEcon.iosam package from this file.

Prerequisites

  • Package gEcon.

Download version 0.2 from R-forge

  • gEcon.iosam_0.2.0.tar.gz — source code (Linux / Un*x systems)
  • gEcon.iosam_0.2.0.zip — binary version for Windows
  • gEcon.iosam_0.2.0.tgz — binary version for Mac (built under OS X 10.11.6 [El Capitan])
