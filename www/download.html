gEcon package

Please note that gEcon is NOT DISTRIBUTED THROUGH CRAN. You have to download the .zip (Windows) or .tag.gz (Linux / Un*x) file, save it on your disk, and install gEcon package from this file.

Download from R-forge:

  • gEcon_0.7.0.tar.gz [884 kB] — source code (Linux / Un*x systems)
  • gEcon_0.7.0.zip [2.0 MB] — binary version for Windows

Important note to the users of previous versions: models written for versions prior to 0.7.0 will not work in the same way or may even fail. Changes include:

  • shocks listed in the .gcn should be now separated by commas (,) instead of colons (;),
  • gEcon inserts Lagrange multipliers for constraints automatically — explicitly declared Lagrange multipliers will not be automatically selected for reduction, and declarations of Lagrange multipliers on time aggregators in static problems will cause an error.
In order to force backward compatibility use backwardcomp option in the options section of your .gcn model file.
R interface has also been slightly changed. In particular shock list provided as an argument to functions: random_path, compute_irf, simulate_model and gecon_simulation class constructor should now be called shock_list instead of eps_list. Please adjust your R code accordingly.
